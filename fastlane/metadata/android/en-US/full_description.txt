Droidbert is a native Android app for reading historical and latest Dilbert comics.

Instead of embedding a web page, it loads comic images directly through the Daily Dilbert API and presents them in a clean, fast reader UI.

Features:
- Native comic reader optimized for phones
- Latest comic shortcut
- Date picker to jump to a specific date
- Previous/Next navigation through available comics
- GIF image decoding for original comic format
- Configurable API endpoint in Settings
- Automatic fallback for hosts that serve comics via get_comics.php + /comics/... paths
- No account required

Default API endpoint:
https://dilbert.xo.nl/api/comic.php
